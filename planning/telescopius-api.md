# Telescopius API

## Source
- URL: https://api.telescopius.com/
- Python SDK: https://pypi.org/project/telescopius-api/
- Site: https://telescopius.com

## Description
API pour la planification d'observations astronomiques :
- Recherche de cibles astronomiques
- Planification de sessions d'observation
- Calcul de visibilité selon la position géographique
- Information sur les objets du ciel profond
